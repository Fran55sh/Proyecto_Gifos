# GIFOS

Proyecto 2 realizado para la carrera de Desarrollo Web Full Stack de Acámica

Consiste en la creación de una Web App para compartir, guardar a favoritos, crear y subir a Giphy.com gifs animados.

Tecnologías utilizadas

- HTML
- CSS
- SASS
- Javascript

API y librería utilizadas

- Giphy API - https://developers.giphy.com/)
- RecordRTC - https://recordrtc.org/)

Link del sitio https://fran55sh.github.io/Proyecto_Gifos/
